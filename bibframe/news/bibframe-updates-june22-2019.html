Updated MARC-to-BIBFRAME Conversion Tools and BIBFRAME Editor

MARC-to-BIBFRAME Conversion Specifications and Conversion Programs Updated

The Library of Congress Network Development and Marc Standards Office (NDMSO) has released a new version of the MARC-to-BIBFRAME conversion specifications and Version 1.5.0  of the marc2bibframe2 conversion program.

  • Download the specifications at: http://lcnetdev.github.io/bibframe/mtbf/
  • Download the conversion programs at https://github.com/lcnetdev/marc2bibframe2

Selected features in this update include:

  • additional URIs for controlled vocabulary terms from MARC tags 007 and 008
  • creation of bf:qualifier elements from parenthetical data in identifier fields
  • revised processing for series statements
  • a change in assignment of ISSNs and Key Titles (was Instances, now Works) to align with IFLA-LRM

LC’s BIBFRAME Editor Updated

In addition, version 1.2.1 of the BIBFRAME editor was released at the end of May.

  • Download the new editor at: https://github.com/lcnetdev/bfe

This version includes software fixes that were reported by participants in the BIBFRAME cataloging pilot at the Library of Congress. The updated editor and the latest cataloging profiles can also be used from this website.
