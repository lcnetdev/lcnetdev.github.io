Language of Parts
BIBFRAME Topics and Issues

Back to Topics and Issues Home

A Work  might have different parts with different languages, for example a book written in English might have a foreword written in French.  BIBFRAME currently handles this by allowing the property bf:language to indicate what part of the Work the language applies to: property bf:languageOfPart is a property of bf:language.  It’s value is a string, for example ‘foreword”.

It has been proposed that a part of a work that needs to be addressed in this manner be treated as a resource.

 The parts could be linked to/from the Work by properties hasPart and isPartOf. hasPart could have specific subproperties for the part types, like hasForeword.

Example:

<http://bibframe.example.con/works/books/goldfinch> a bf:Work ;
      bf:title "The Goldfinch" ;
      bf:language  <http://id.loc.gov/vocabulary/iso639-1/en>
      bf:hasForeword <http://bibframe.example.con/workParts/forewords/goldfinch> .

<http://bibframe.example.con/workParts/forewords/goldfinch> a bf:Foreword ;
      bf:ispartOf  < http://bibframe.example.con/works/books/goldfinch >.
      bf:language  <http://id.loc.gov/vocabulary/iso639-1/fr> .
