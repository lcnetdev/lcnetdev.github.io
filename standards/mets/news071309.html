METS Namespace Information Page and URI Resolution Service (July 13, 2009)


Thanks to the efforts of Clay Redding and Glenn Gardner at Library of Congress, a new METS Version 1.x Namespace information page is now available along with an accompanying version 1 namespace URI resolution service.

The Version 1.x Namespace information page provides information about mets version 1.x schemas governed by the namespace URI:
"http://lcnetdev.github.io/METS/".

Users can get to the namespace information page via the new namespace page resolution service. This resolution service works as follows. If a user agent attempts to resolve the version 1 namespace URI (http://lcnetdev.github.io/METS/) and specifies "text/html" as the desired MIME type in the http Accept request header, the agent will receive the namespace information page. This happens by default with most common web browsers like IE, Firefox, etc. However, if the user agent specifies "text/xml" as the desired MIME type in the http Accept request header, the agent will receive the most recent incremental release of the version 1 METS schema. Requesting non-default MIME types often requires special tools, although these are often found in XML and other programming authoring environments.

METS website users can also get to namespace information page via a new link under the "Current Schema Version and Documentation" header on the "METS Schema & Documentation" page:
( http://lcnetdev.github.io/standards/mets/mets-schemadocs.html ).

Rick Beaubien
