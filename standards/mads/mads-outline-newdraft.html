Outline of Elements and Attributes in MADS (Mar. 10, 2005)

This document contains a listing of elements and their related attributes in the Metadata Authority Description Schema (MADS) with values or value sources where applicable. It is an "outline" of the schema.

In a MADS record, there must be at least one element under <authority>. There may be only one <authority>, but elements may be used together under an <authority> (for instance a <name> and <titleInfo> or <topic> and <geographic>). All other elements and attributes are optional. Attributes are not in a mandated sequence and not repeatable (per XML rules).
