audioMD, videoMD, and AES Schemas and Documentation

The audioMD and videoMD schemas were originally created at the Library of Congress in 2000-2003 as part of the Audio-Visual Prototyping project. In 2011, the Audio Engineering Society published  two new metadata standards for audio materials--AES57-2011 and AES60-2011.

Metadata for Audio

Two new metadata standards for audio materials were published in 2011 by the Audio Engineering Society -- AES57-2011 and AES60-2011.

AES Schema

  • AES57-2011, AES Standard for Audio Metadata - Audio Object XML Schema
  • AES60-2011, AES Standard for Audio Metadata - Core Audio Metadata XML Schema

AES Documentation

  • AES57-2011, AES Standard for Audio Metadata - Audio Object Structures for Preservation and Restoration

videoMD

Users should note that an effort is under way in the Federal Agencies Digitization Guidelines Working Group to develop a video metadata specification during 2011-2012, similar to videoMD but with some additional elaborations.
