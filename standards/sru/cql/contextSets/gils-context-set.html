GILS Context Set

This is the CQL context set for GILS (Global Information Locator Service).   See also the GILS Profile.

The identifier for this context set is:info:srw/cql-context-set/14/gils-v1.0

 

Indexes

The names and definitions of indexes follow the guidance given in the CQL Index-naming Convention. This set encompasses three referents: resource (i.e., "information resource"), product (i.e., "information resource product"), and record (i.e., "locator record of the information resource"). The implicit referent is "resource".

The term information resource in this context is defined as: "The means to supply or provide information, including products or services derived from the information resource."

Examples

  1. To retrieve records about information resources having "USGS" in the point of contact and modified since 1 January 2004, the query would be:

gils.contact = "USGS" and rec.lastModified > "2004-01-01"
