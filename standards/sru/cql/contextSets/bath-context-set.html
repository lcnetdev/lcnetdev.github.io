The bath Context Set for CQL

Version 1.0

The bath context set defines several indexes to support the Z39.50 Bath Profile. There is also an SRU Bath Profile.

  • URI for this context set: http://zing.z3950.org/cql/bath/2.0/
  • Recommended short name: bath

This context set defines indexes only. The indexes defined are in the following table.

 

Index Reference in Bath Profile
keyTitle 5.A.2.1-5
possessingInstitution 5.A.2.10
name 5.D.1.1-4
personalName 5.D.2.1-4
corporateName 5.D.2.5-8
conferenceName 5.D.2.9-12
uniformTitle 5.D.2.13-17
issn 5.D.2.18
geographicName 5.D.2.20-23
notes 5.D.2.27-28
topicalSubject 5.D.2.31-35
genreForm 5.D.2.36-40
