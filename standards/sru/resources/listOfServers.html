France

French National Library

BnF provides an SRU server (1.2 version) which give access to all BnF catalog metadata (bibliographic and authority).

Inquiry form for this SRU service       documentation        explain      Criteria

This service is open to every one without any restrictions.

Sample searches

  • dinosaures within bibliographic records: http://catalogue.bnf.fr/api/SRU?version=1.2&operation=searchRetrieve&query=bib.anywhere%20all%20%22dinosaures%22

  • documents from Gallmeister publisherwithin the BnF catalog (in dublin core): http://catalogue.bnf.fr/api/SRU?version=1.2&operation=searchRetrieve&query=bib.publisher%20all%20%22Gallmeister%20%22&recordSchema=dublincore

  • authority records for Persons like "Wolfgang Amadeus Mozart" (with a style sheet): http://c
  • bnf.fr/api/SRU?version=1.2&operation=searchRetrieve&query=aut.accesspoint%20all%20%22wolfgang%20amadeus%20mozart%22%20and%20aut.type%20all%20%22pep%22&stylesheet=xslt/IXMn_HTML.xsl

  • "Charlie Chaplin" authority record using its ISNI identifier: http://catalogue.bnf.fr/api/SRU?version=1.2&operation=searchRetrieve&query=aut.isni%20all%20%220000000121213863%22
