The PREMIS maintenance activity is responsible for maintaining, supporting, and coordinating future revisions to the PREMIS data dictionary. The Preservation Metadata: Implementation Strategies Working Group, convened by the Online Computer Library Center (OCLC) and the Research Libraries Group (RLG), initially developed the PREMIS data dictionary as a specification with the goal of creating an implementable set of "core" preservation metadata elements, with broad applicability within the digital preservation community. A supporting XML schema allows for implementation of the core metadata element set and is maintained in the Network Development and MARC Standards Office of the Library of Congress.
