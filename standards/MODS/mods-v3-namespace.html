December 2008

This is the XML namespace document for MODS version 3 (for MODS 3.x versions 3.4 and higher). 

The namespace identifier for this namespace is "http://www.loc.gov/mods/v3"

  Listed below are the names of the elements, attributes, and type definitions in  this namespace, which may be referenced by external schemas.

Known XML schemas that use names from this namespace:

  • MODS draft version 3.4
