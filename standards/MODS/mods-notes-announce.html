Announcement: MODS Note Types Now Available

A list of MODS note types used by MODS implementors is now available at: www.loc.gov/standards/mods/mods-notes.html. The types are used as values in the MODS <note> element with the attribute "type." The list includes the source of the note type and its MARC equivalent, if applicable.

The MODS Note Types list is not exhaustive and includes note types that have been contributed by some implementors. Its purpose is to promote interoperability between MODS records, although the note type list remains uncontrolled. The Library of Congress welcomes contributions to the list by implementors.
