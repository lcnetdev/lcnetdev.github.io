Outline of Elements and Attributes in MODS

This document contains a listing of elements and their related attributes in MODS with values or value sources where applicable. It is an "outline" of the schema.

All elements (except titleInfo with subelement title) and attributes are optional. Attributes are not in a mandated sequence and not repeatable (per XML rules). "Set" below means the subelements may occur in any order; "ordered" below means the subelements must occur in the order given; "choice" means only one subelement from the list may be selected (if another is needed, the primary element is repeated). All elements and subelements are repeatable except where noted.

1. titleInfo [mandatory]
  Subelements [set]:
    title [mandatory]
    partNumber
    partName
    nonSort
  Attributes:
    ID
  type (abbreviated, translated, alternative, uniform)
  authority (see lcnetdev.github.io/marc/sourcecode/authorityfile)
  displayLabel
  xlink [to uniform title authority record]

2. name
  Subelements [ordered]:
    namePart
      Attribute: type (date)
    displayForm
    affiliation
    role
      Attribute: authority [no controlled list]
    description
  Attributes:
    ID
    type (personal, corporate, conference)
    authority (see lcnetdev.github.io/marc/sourcecode/authorityfile)
    xlink [to name authority record]

3. typeOfResource [list in schema]
  Subelements:
    [none]
  Attributes:
    collection (yes)
    manuscript (yes)

4. genre
  Subelements:
    [none]
  Attributes:
    authority (see lcnetdev.github.io/marc/sourcecode/genre) [MARC list commented in schema]

5. publicationInfo
  Subelement [ordered]
    placeCode
      Attribute: authority (marc, iso3166)
    place
    publisher
    dateIssued
      Attributes: encoding (w3cdtf, iso8601, marc); point start, end)
    dateCreated
      Attributes: encoding (w3cdtf, iso8601, marc); point (start, end)
    dateCaptured
      Attributes: encoding (w3cdtf, iso8601, marc); point (start, end)
    edition
    issuance (continuing, monographic)
    frequency
  Attributes
    [none]

6. language
  Subelements:
    [none]
  Attributes:
    authority (rfc3066, iso639-2b)

7. physicalDescription
  Subelements [set]:
    form (braille, electronic, microfiche, microfilm)
    internetMediaType
    extent
    digitalOrigin (born digital, reformatted digital)
    note
  Attributes:
    [none]

8. abstract
  Subelements:
    [none]
  Attributes
    xlink
    xml:lang
    type

9. tableOfContents
  Subelements
    [none]
  Attributes:
    xlink
    xml:lang
    type

10. targetAudience
  Subelements [choice]:
    listValue authority (see lcnetdev.github.io/marc/sourcecode/target) [MARC list commented in schema]
    otherValue [no controlled list]  
  Attributes:
    [none]  

11. note
  Subelements:
    [none]
  Attributes:
    xlink
    xml:lang
    type

12. cartographics
  Subelements [ordered]:
    coordinates
    scale [not repeatable]
    projection [not repeatable]
  Attributes:
    [none]

13. subject
  Subelements [set]:
    topic
    geographic
    temporal
      Attributes: encoding (w3cdtf,iso8601, marc); point (start,end)
    titleInfo [see titleInfo]
    name [see name]
    hierarchicalGeographic [set]
      continent
      country
      province
      region
      state
      territory
      county
      city
      island
      area
  Attributes:
    ID
    authority (see lcnetdev.github.io/marc/sourcecode/subject)
    xlink

14. classification
  Subelements:
    [none]
  Attributes:
    authority (see lcnetdev.github.io/marc/sourcecode/classification)
    edition [no controlled list]

15. relatedItem
  Subelements [ordered]:
    titleInfo [see titleInfo]
    name [see name]
    physicalDescription [see physicalDescription]
    identifier [see identifier]
    note [see note]
  Attributes:
    ID
    type (preceding, succeeding, reproduction, original, host, constituent, series, related)
    xlink [to record for related item]

16. identifier
  Subelements:
    [none]
  Attributes:
    type [commented list in schema: doi, isbn, isrc, ismn, issn, issue number, istc, lccn, local, matrix number, music publisher, music plate, sici, uri, upc, videorecording identifier]

17. location
  Subelements:
    [none]
  Attributes:
    authority (see lcnetdev.github.io/marc/sourcecode/organizations)

18. accessConditions
  Subelements:
    [none]
  Attributes:
    xml:lang
    xlink
    type

19. extension
  Subelements:
    [none]
  Attributes:
    [none]

20. recordInfo [not repeatable]
  Subelements [ordered]:
    recordContentSource
    recordCreationDate
  Attributes: encoding (w3cdtf, iso8601, marc); point (start, end)
    recordChangeDate
  Attributes: encoding (w3cdtf, iso8601, marc); point (start, end)
    recordIdentifier
  Attribute: source [no controlled list]
  Attributes:
    [none]
