Announcement: MODS Version 3.1 Available for Use

The revised MODS schema version 3.1 is now available online at: lcnetdev.github.io/mods/v3/mods-3-1.xsd. The Outline of elements and attributes reflect version 3.1. The MARC mappings will be revised shortly, although the impact on them is limited, since the changes to the schema are fairly minor and many of the elements added do not have MARC equivalents. There is also minimal impact on the MARCXML to MODS stylesheet, guidelines, and examples; these will also be revised to reflect the new schema shortly. There is no need to convert existing 3.0 records, so there is no need for a stylesheet for that purpose.

Changes from the previous version 3.0 are documented at: http://lcnetdev.github.io/mods/changes-3-1.html.

The revision is based on input from implementors and from others who have participated in this list. Thank you all for contributing.
